#ModResume is a Resume organizer.
* Add your experience.
* Tag as you go.
* When you need a new resume, filter by tag, fine tune and send!

##Tech Stacks
* Rails
* Javascript
* JQuery UI
* RSpec / Capybara
* Tiny MCE
* Devise
* Sendgrid
* Heroku
